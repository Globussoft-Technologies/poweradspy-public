# PowerAdSpy

Public repository for **PowerAdSpy** — an ad-intelligence ("ad spy") platform for tracking and reverse-engineering competitor ads across Facebook, Instagram, YouTube, Google and 9+ other networks.

## About

PowerAdSpy is a commercial product by Globussoft Technologies. It lets marketers and advertisers search, filter and analyze live and historical ads across major networks to research creatives, spot trends and model winning campaigns.

- Website: https://poweradspy.com

## This repository

This is the public-facing repository for PowerAdSpy (public resources, documentation and issue tracking). The product's application code is maintained in private repositories.
